1.
Download this tool:
https://bluetoothinstaller.com/bluetooth-command-line-tools

2.
Install it, and allow it to be added as a Path variable.

3.
Run 'btdiscovery' in cmd.

4.
Copy your device's MAC address.

5.
Now replace 'MAC' word mentioned in both batch files (i.e. connect.bat and disconnect.bat) with the address you copied.

Done!
